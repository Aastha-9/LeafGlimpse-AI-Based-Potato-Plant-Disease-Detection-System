# 🌿 LeafGuard – AI-Based Plant Disease Detection System

LeafGuard is an AI-based plant disease detection system that uses Convolutional Neural Networks (CNN) to identify plant leaf diseases from images and support early diagnosis.

## 📂 Project Structure
- **api/** – Backend APIs for image upload and prediction
- **frontend/** – Web interface for user interaction
- **models/** – Trained CNN model files
- **Training/** – Model training and experimentation scripts

## 🛠 Technologies Used
- Python
- TensorFlow / Keras
- CNN
- Image Processing
- Flask / FastAPI (Backend)
- HTML, CSS, JavaScript (Frontend)

## 📌 Project Status
Ongoing – Final Year Project

## 🚀 Future Scope
- Accuracy optimization
- Multilingual disease recommendations
- Web/Desktop deployment
